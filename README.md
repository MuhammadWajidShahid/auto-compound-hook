# Auto Compound Hook

Auto Compound is a hook which is designed to reinvest the earned fee back to lp position. This allows users to earn more yield on their lp positions.

## Usage

### Build

```shell
$ forge build
```

### Test

```shell
$ forge test
```

### Format

```shell
$ forge fmt
```
